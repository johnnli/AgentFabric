# AgentFabric
A light-weighted tool for data fabric implemention.
# Get started
## What's this?
Tiny auto agents controlled by the MIND.json, help to extract data, calculating and representation.
## How to use it?
1. Server side (web_app.py) deployment
2. Edit MIND.json
3. Run app.py

That's it~
